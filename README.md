# Currency Checker

A responsive Currency Checker web application built with HTML, CSS, and JavaScript. The application fetches live exchange rates using a public API and allows users to convert currencies instantly.

## Features

- Live exchange rates
- Currency conversion
- Swap currencies
- Popular exchange rates
- Responsive UI
- Dark theme
- Error handling
- Fast and lightweight

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Exchange Rate API

## How to Run

1. Clone the repository

```bash
git clone https://github.com/manoj-codes27/CurrencyChecker.git
```

2. Open `index.html`

No installation required.
